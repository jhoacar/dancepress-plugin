== Description ==

DancePress offers a complete solution for dance schools looking for an attractive WordPress website integrated with a powerful Dance School Management and Ecommerce Application.

Simplify common tasks, save time and money, concentrate on what you do best.

=== Complete Solution for Dance Schools ===

DancePress offers all the features of more expensive Dance School management software, but integrated into Wordpress. That means you can run all aspects of your school's back office through the plugin, and integrate it with your existing Wordpress site. By upgrading to pro or enterprise editions, you can communicate with clients, parents and students, provide simple and secure online registrations, and offer with a wide range of price structures, installments, and more.

__Installed in seconds - easy to use__

= Features =

* Manage every aspect of your dance studio through an intuitive interface.
* Manage client/student details.
* Clients build their own custom class schedules, based on their age and skill levels at registration.
* Handle competitive and recreational students separately.
* Create classes, assign students, monitor attendance, handle costumes and sizes.
* Communicate with single clients, entire classes, or custom groups. [Pro/Enterprise feature]
* Accept secure online payments and handle hundreds of scheduled payment installments in seconds. [Pro/Enterprise feature]
* Keep a complete history of past years/sessions.
* Provide a private ‘parent portal’ for school announcements and news. [Pro/Enterprise feature]
* Monitor enrollments, set class limits, follow drop-outs.
* Fully integrated with WordPress. Run your website and dance school all from the same software.
* Community support or pro support options with paid upgrades.

== Installation ==

DancePress Community Edition can be installed direct from Wordpress:

1. Go to Plugins > Add New in your Wordpress dashboard and search for 'DancePress'.
2. Click 'Install Now' and then 'Activate'.
3. Follow the instructions to complete some basic required information.
4. Start using DancePress!.

Alternatively:
1. download the latest plugin from the [Wordpress Plugin site](https://wordpress.org/plugins/dancepress-trwa/).
2. Unzip the archive and copy the files to the /wp-content/plugins/ directory of your installation.
3. Activate the plugin.
4. Follow the instructions to complete some basic required information.
5. Start using DancePress!
